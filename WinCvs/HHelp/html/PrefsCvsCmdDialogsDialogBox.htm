Command Dialogs Preferences dialog box

The dialog options allow you to control global options of WinCvs:

Skip dialogs

Select to avoid displaying settings dialogs for commands. To force those dialogs press the force key when invoking command.

Key to force the dialog

Select the key to force the dialog marked to be skipped. Shift key is a recommended choice to allow shortcut keys operated with Ctrl key to work.

Dialog confirmation (for multiline editing only)

Select the dialog confirmation method when editing multiline entry such as commit log message. If Ctrl + Enter is selected to confirm the dialog then the Enter key will provide line-break and vice-versa.
