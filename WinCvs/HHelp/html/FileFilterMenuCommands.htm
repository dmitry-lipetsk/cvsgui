File Filter menu commands

Use this menu commands to display and hide the files and folders matching specified file filters such as commitable, unknown and ignored files. A check mark appears next to the file filter item when the filter is selected.

The File Filter menu offers the following commands:

Clear All Clear All Filters.
Show All Commitable Show All Commitable Files Only.
Show Changed Show Changed Files Only.
Show Added Show Added Files Only.
Show Removed Show Removed Files Only.
Show Conflicts Show Conflicts Files Only.
Show Missing Show Missing Only.
Hide Missing Hide Any Missing.
Show Unknown Show Unknown Only.
Hide Unknown Hide Any Unknown.
Show Ignored Show Any Ignored.
