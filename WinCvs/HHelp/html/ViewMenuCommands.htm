View menu commands

The View menu offers the following commands:

Refresh Refresh view.
Flat Mode Shows files in all subfolders or in selected folder only.
Toolbars menu Toolbars menu.
Status Bar Shows or hides the status bar.
Workspace Show or hide the workspace bar.
Output Show or hide the console bar.
Browse Location menu Browse Location menu.
File Filter menu File Filter menu.
Filter Bar menu Filter Bar menu.
