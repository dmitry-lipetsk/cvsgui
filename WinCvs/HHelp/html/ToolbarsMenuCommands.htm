Toolbars menu commands

Use this menu commands to display and hide the Toolbars, which include buttons for some of the most common commands in WinCvs, such as Update, Commit or Graph. A check mark appears next to the menu item when the Toolbar is displayed.

The Toolbars menu offers the following commands:

Standard Show or hide the standard toolbar.
Browse Show or hide the browser bar.
Files Show or hide the files bar.
Multi-Users Show or hide the multi-users bar.
Tags Show or hide the tags bar.
Filter Show or hide the filter tool bar.

See Toolbar for help on using the toolbar.
